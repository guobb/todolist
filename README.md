# todolist  项目命令
### 安装express 生成器

> npm install -g express-genertor

### 生成项目

> express -e todolist

### 安装bower命令行工具

> npm install -g bower

### 初始化bower配置文件

> bower init 

### 在项目下添加配置文件.bowerrc

> {"directory":"./public/lib"}

### 安装前端框架

> bower install jquery --save

> bower install angular -- save

> bower install bootstrap --save
